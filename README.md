# Employee-Appraisal
Simple MEAN stack employee appraisal/rating management  

## Instructions
- Add mongodb database url in backend/config/dev.env
- Install node modules using 
  ```
   npm i
  ```
- To start Express server 
  ```
   npm run dev 
  ```
- To start angular dev server
  ```
  ng serve
  ```
- Open in browser
  ```
  localhost:4200
  ```
